0.1.0 (2013-04-03)

Initial public release

Implement Keystone, Glance, Nova, Neutron integration in order to provide a working API/UI for the following features:  
- Login/Logout  
- List Images  
- List Servers  
- Show Server  
- Create/Delete Server  
- List Floating IPs  
- Add Floating IP  
- Start/Stop/Resume/Reboot Server  
- List Keypairs  
- Add Keypair  
- List Networks  
- Add Network/Subnet  
- Show Quotas
